## Project Goal

Develop a professional Telugu Astrology Application based on Tritha Siddhanta and modern astronomical calculations.

The application shall:

- Calculate Horoscope
- Generate Panchangam
- Display South Indian Kundali
- Calculate Dasa
- Calculate Gocharam
- Generate Telugu Reports
- Store Client Records
